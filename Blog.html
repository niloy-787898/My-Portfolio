Html- Base of Web desighn

HTML is referred to as the foundational language of the web. It is responsible for the basic structure of the web pages. It creates the skeleton of the web page. This mark- up language basically consists of tags and elements that helps in building a website or a blog from scratch. HTML is a crucial language and constitutes the basic structure of all modern websites out there. If you wish to or want to enter the world of web development, HTML is a mandatory language.

Css - Styling a website

CSS stands for Cascading Style Sheets It describes how HTML elements are to be displayed on screen, paper. It can control the layout of multiple web pages all at once External stylesheets are stored in CSS files

Javascript-heart of a website

JavaScript runs on the client side of the web. It can be used to design / program how the web pages behave on the occurrence of an event. It is an easy to learn and also powerful scripting language. It is widely used for controlling web page behavior.

Node.js-Back end booster of a website

Node.js runtime environment has been ranked the technology most commonly used by professional developers. It is an event-driven JavaScript runtime. It has myriad potential uses for JavaScript development. It has a great environment for building efficient network applications.
